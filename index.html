👋 Welcome to ODR!

Think of this app as your magic pocket-factory. You can ask Artificial Intelligence to write website code for you, paste it into the CODE tab, see what it looks like in the PREVIEW tab, and instantly put it on the real internet using the DEPLOY tab.

Step 1: Get a Free "Storage Unit"

To put a website on the internet for free, we use a service called GitHub. It's like a free public storage unit for your code.

  • Go to GitHub.com and sign up for a free account.
  • Remember your Username. You will need to type this into the DEPLOY tab later!

Step 2: Get your Security Key

For this app to send code to your GitHub account securely, it needs a special digital key called a Personal Access Token.

1. Log into GitHub and click your profile picture (top right).

2. Click Settings at the bottom of the menu.

3. Scroll all the way down the left menu and click Developer Settings.

4. Click Personal access tokens, then click Tokens (classic).

5. Click Generate new token (classic).

6. Give it a name (like "ODR App"). Check the box that says repo (this gives it permission to save files).

7. Scroll down and click Generate. Copy the long password it gives you! Paste this into the "Access Token" box in the DEPLOY tab.

Step 3: Create a Folder for your Website

Now we need a specific folder inside GitHub to hold your website files. This is called a Repository.

  • On GitHub, click the + icon at the top and select New repository.
  • Give it a short, simple name (e.g., my-website). Type this exact name into the "Repository Name" box in the DEPLOY tab.
  • Make sure it is set to Public. Scroll down and click Create repository.

Step 4: Turn the Internet On!

The last step is telling GitHub to treat that folder as a real website instead of just a storage box.

  • Inside your new repository on GitHub, click the Settings tab at the top.
  • On the left menu, click Pages.
  • Under "Build and deployment" -> "Source", select Deploy from a branch.
  • Under "Branch", change "None" to main, and click Save.

🎉 You're Ready!

You only ever have to do that setup once! From now on, just paste your code into the CODE tab, hit DEPLOY TO GITHUB PAGES, and your website will update live on the internet!
